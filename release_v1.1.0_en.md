# Release Notes - v1.1.0 🎵

We are excited to announce the release of **v1.1.0**, focusing on personalized music management and a significantly smoother browsing experience.

## ✨ New Features

### 🎧 User Playlists
- **Full Playlist Management:** Create, view, and delete your own custom playlists from the dedicated **Playlists** tab.
- **Playlist Details:** Explore tracks inside any playlist with the new Detail view.
- **Integrated Creation:** Create a new playlist directly from the Now Playing screen while adding a song.
- **Supabase Backend:** All playlists are securely synced to your account via Supabase.

### 🔍 Enhanced Explore Experience
- **Artist Filtering:** Tap on any artist to see only their tracks.
- **Universal Search:** Quickly find any track in the "All Songs" library.
- **Categorized Browsing:** Seamlessly switch between personalized recommendations, artist lists, and the full catalog.

## 🚀 Performance & UI Improvements

- **Native Lazy Loading:** Migrated the "All Songs" list to `FlatList`, enabling buttery-smooth scrolling even with thousands of tracks.
- **Refined Navigation:** Improved route handling for dynamic playlist views and player integration.
- **TypeScript Optimization:** Strengthened type definitions across the app to prevent runtime errors.
- **Bug Fixes:** Resolved issues with image loading and icon visibility in the Explore tab.
